# Random Task Generator

Пасечник Игорь Павлович

## Описание

Приложение «Random Task Generator» — это графический инструмент для генерации случайных задач с возможностью фильтрации по категориям и сохранением истории.

**Основные функции:**
* Генерация случайных задач по выбранным категориям (учёба, спорт, работа).
* Фильтрация задач по типу.
* Сохранение истории сгенерированных задач в JSON‑файл.
* Отображение истории в графическом интерфейсе.
* Проверка корректности ввода (не пустая строка при добавлении задач).
